## 开始 Clover 之旅

```shell
composer create-project clovers/clover --prefer-dist
cd clover && php -S 127.0.0.1:8888 -t public
```
or

```shell
git clone https://github.com/donghaichen/clover.git
cd clover
composer intall
php -S 127.0.0.1:8888 -t public
```
