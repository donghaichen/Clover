## 开始Clover之旅

```shell
composer create-project clovers/clover --prefer-dist
cd clover/public && php -S 127.0.0.1:3000
```
or


```shell
git clone https://github.com/donghaichen/Clover.git
cd clover
composer intall
cd public && php -S 127.0.0.1:3000
```
